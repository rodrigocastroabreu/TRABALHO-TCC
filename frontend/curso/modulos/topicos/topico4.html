Payoff

Resultado ou retorno de cada decisão

O que é Payoff?

Payoff (também chamado de recompensa, retorno ou lucro) é o resultado que um jogador recebe ao final de um jogo, dependendo das estratégias escolhidas por ele e pelos outros jogadores.

Características Principais

  1. Pode ser positivo (ganho) ou negativo (perda)
  2. Depende das decisões de TODOS os jogadores
  3. Motiva as decisões dos jogadores
  4. Pode ser financeiro, pontos, satisfação, etc.

Matriz de Payoff

A matriz de payoff é uma tabela que mostra todos os possíveis resultados de um jogo para todas as combinações de estratégias.

Exemplo: Jogo do Rock-Paper-Scissors

Você ↓ / Outro → Pedra Papel Tesoura
Pedra 0, 0 -1, 1 1, -1
Papel 1, -1 0, 0 -1, 1
Tesoura -1, 1 1, -1 0, 0

Formato: (Seu payoff, Payoff do outro)

Função de Utilidade

A utilidade é uma medida numérica que representa a satisfação ou preferência de um jogador por um resultado.

Princípios Importantes

  1. Cada jogador pode ter uma função de utilidade diferente
  2. A utilidade pode não ser só dinheiro (inclui riscos, preferências, etc.)
  3. Payoff é o resultado; utilidade é a satisfação com aquele resultado
  4. Algumas pessoas preferem risco; outras preferem segurança

Exemplo: Você ganha R$100. Para uma pessoa rica é 10 de utilidade; para uma pessoa pobre é 200 de utilidade!

Exemplos de Payoff em Contextos Reais

  1. 💼 Negociação: Payoff é o preço acordado
  2. ⚖️ Tribunal: Payoff é a sentença (prisão, multa, etc.)
  3. 🌳 Meio Ambiente: Payoff inclui poluição (negativo)
  4. 🤝 Cooperação: Payoff maior quando ambos cooperam
  5. 📱 Tecnologia: Payoff é participação de mercado
