Dilema do Prisioneiro

Exemplo clássico de conflito em teoria dos jogos

O Problema Clássico

Dois suspeitos são presos por um crime. Cada um está em uma cela separada e não podem se comunicar.

A Situação

Cada prisioneiro tem duas opções:

  1. Confessa: "Denunciar o outro"
  2. Cala-se: "Não dizer nada"

Regra: Se um confessa e o outro não, quem confessou fica livre (0 anos) e o outro pega 3 anos. Se ambos confessam, cada um pega 2 anos. Se ambos se calam, cada um pega 1 ano.

Análise do Equilíbrio

Matriz de Payoff

A\B Confessa Cala-se
Confessa -2, -2 0, -3
Cala-se -3, 0 -1, -1

Payoff são anos de prisão (números negativos = piores)

O Paradoxo

Se ambos se calam → cada um pega -1 (MELHOR para ambos)

Mas se o outro se cala, confessar → você fica livre (0) vs -1!

Por isso ambos confessam → Equilíbrio de Nash em (-2, -2), mesmo sendo pior para ambos!

Variações Importantes

  1. 🔄 Dilema Iterado: Jogo repetido muitas vezes (estratégias como "Tit-for-Tat")
  2. 👥 N-Prisioneiros: Mais de 2 jogadores
  3. 🎮 Bem Público: Cada um contribui ou não a um fundo comum
  4. 🤐 Assimétrico: Diferentes payoffs para cada prisioneiro
  5. 💬 Com Comunicação: Jogadores podem se comunicar (mas não enforçar acordo)

Aplicações Reais do Dilema

O Dilema do Prisioneiro explica por que pessoas racionais frequentemente cooperam menos que o ideal:

  1. 💼 Cartéis: Empresas que combinam preços altos (incentivo para trair)
  2. 🌍 Clima: Países que emitem CO₂ (todos sofrem mas individualmente têm incentivo)
  3. 🏋️ Doping: Atletas que usam substâncias (todos melhoram o resultado se todos usarem)
  4. 📱 Redes Sociais: Espalhamento de informações falsas (viral rápido demais)
  5. 🏛️ Negociação: Países que desarmam primeiro ficam vulneráveis
  6. 🏪 Promoções: Lojas competem com preços cada vez mais baixos
